===== DATABASE QUERY TO FIND UNTRACKED CHANGES =====

SELECT r.id, r.ticket_id, r.status, 
       COUNT(rd.id) as disposition_count,
       r.updated_at, r.created_at
FROM reports r
LEFT JOIN report_dispositions rd ON r.id = rd.report_id
WHERE r.status != 'pending'  -- likely changed
AND rd.id IS NULL  -- but no disposition record
GROUP BY r.id
ORDER BY r.updated_at DESC;

This query finds reports that:
1. Have a non-pending status (changed from default)
2. Have NO disposition records
3. Therefore, have untracked status changes

Expected results: ~31 reports
